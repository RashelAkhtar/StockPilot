# StockPilot

StockPilot is a full-stack inventory and sales management dashboard designed for small businesses.
It enables shop owners to manage products, record multi-item sales, and track revenue and profit trends in real time through an intuitive analytics dashboard.

Built with **React, Express, and PostgreSQL**, the app combines operational workflows with visual insights to simplify day-to-day business management.

---

## Features

* Product inventory management
* Multi-item sales recording with cart workflow
* Customer order history tracking
* Revenue and profit analytics dashboard
* Sales trend and profit trend charts
* Top and least sold product insights
* Low-stock alerts
* Secure authentication with JWT cookies

---

## Tech Stack

**Frontend**

* React
* Vite
* React Router
* Chart.js
* TanStack Table

**Backend**

* Node.js
* Express
* PostgreSQL
* JWT Authentication
